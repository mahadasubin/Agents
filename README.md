# LinkedIn AI Post Agent

## Setup
1. Clone repo

2.cd Python_Agent
    -python3 -m venv venv
    -source venv/bin/activate      # if on macOS/Linux
    -pip install -r requirements.txt
    -python -m uvicorn main:app --reload --port 8000
   
3.cd server
    -npm install
    -npm run dev

4.cd client
    -npm install
    -npm run dev

5.open http://localhost:5173
